Transcription Assistant User Guide

🎵 Quick Start

Upload Audio - Click the upload area or drag & drop your MP3/WAV file into the page
Playback Control - Click the big play button or press the Space bar to play/pause
⌨️ Keyboard Shortcuts

Table


Key	Function
Space	Play / Pause
←	Rewind 3 seconds
→	Fast forward 3 seconds
A	Set Point A (loop start)
B	Set Point B (loop end)
C	Clear A-B loop
🔄 A-B Loop

How to use:
Play to your desired starting position, press A or click "Set Point A"
Continue playing to your desired ending position, press B or click "Set Point B"
The audio will automatically loop between Points A and B
Press C or click "Clear" to exit the loop
📱 Touch Screen Support

Progress Bar - Press and drag to seek to any playback position
Buttons - All buttons support touch input
💡 Tips

The green marker on the progress bar indicates Point A, the orange marker indicates Point B
Dragging the progress bar automatically seeks to that position
To change audio files, click the "Change File" button in the top right corner



## 扒带小助手 使用说明

### 🎵 快速开始

1. **上传音频** - 点击上传区域或拖拽 MP3/WAV 文件到页面
2. **播放控制** - 点击大播放按钮或按空格键播放/暂停

---

### ⌨️ 键盘快捷键

| 按键 | 功能 |
|------|------|
| `空格` | 播放 / 暂停 |
| `←` | 快退 3 秒 |
| `→` | 快进 3 秒 |
| `A` | 设置 A 点（循环起点） |
| `B` | 设置 B 点（循环终点） |
| `C` | 取消 AB 循环 |

---

### 🔄 AB 循环

**使用方法：**
1. 播放到想要循环的起始位置，按 `A` 或点击「设置 A 点」
2. 继续播放到想要循环的结束位置，按 `B` 或点击「设置 B 点」
3. 音频会自动在 A、B 点之间循环播放
4. 按 `C` 或点击「取消」可退出循环

---

### 📱 触屏支持

**进度条** - 按住并拖动可跳转播放位置
**按钮** - 所有按钮都支持触屏点击

---

### 💡 小贴士

进度条上的绿色标记是 A 点，橙色标记是 B 点
拖动进度条时会自动跳转到对应位置
更换音频文件请点击右上角的「更换文件」按钮
